◆ GetBackgroundChecking()

bool wxAutoExcel::wxExcelErrorCheckingOptions::GetBackgroundChecking ( )

Alerts the user for all cells that violate enabled error-checking rules.

When this property is set to True (default), the AutoCorrect Options button appears next to all cells that violate enabled errors. False disables background checking for errors.
